Differece between var let and const in js

var

A var variable can be declaring particular variable anywhere in html file. It's a global variable. It can be redeclared and updated.
example:
var name="Shan";
console.log(name);
output:shan
var a=10;
console.log(a);
output:10

let

A var variable can be declaring in the scope. It's work inside the scope. It's a local variable. It can be updated but not redeclared.
example:
let p=23;
console.log(p);
output:23
let name="San"; console.log(name);
output:san

const

The value cannot be changed.
example:
const country="India"; console.log(country);
output:India
